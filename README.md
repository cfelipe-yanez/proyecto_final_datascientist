# Proyecto Final - Bootcamp Data Scientist

## Descripción
Este repositorio contiene el Proyecto Final del Bootcamp de Data Scientist de TripleTen. 
El objetivo del proyecto es integrar los conocimientos adquiridos a lo largo del curso y resolver un problema real de negocio utilizando técnicas de ciencia de datos y machine learning.

## Estructura del proyecto
- `etapa_1_plan_de_trabajo/`: Contiene el plan detallado de trabajo con la metodología a seguir, entendimiento del caso y análisis de los datos.
- `etapa_2_codigo_de_solucion/`: Contiene el código de implementación de los modelos predictivos y análisis de datos.
- `etapa_3_informe_de_solucion/`: Contiene el informe final con conclusiones, hallazgos, recomendaciones y storytelling.

## Instrucciones para ejecutar
1. Abrir los notebooks en JupyterLab o Jupyter Notebook.
2. Ejecutar las celdas de código en orden para reproducir resultados.
3. Revisar las celdas de Markdown para entender el razonamiento y las conclusiones.
